# Neues Projekt

## Struktur

```
new-project/
├── index.html          # Einstiegspunkt
├── assets/             # Bilder, Schriften, etc.
└── src/
    ├── css/
    │   └── style.css   # Styles
    └── js/
        └── app.js      # JavaScript Einstiegspunkt
```

## Start

Einfach `index.html` im Browser öffnen, oder einen lokalen Dev-Server starten:

```bash
npx serve .
```
